# Sistema Inteligente de Agua con Asistencia por Voz

---

## Descripción General

Este proyecto consiste en el desarrollo de un sistema físico inteligente orientado a la purificación y dispensado automático de agua, integrando asistencia por voz mediante Inteligencia Artificial.

A diferencia de otros proyectos basados en software o aplicaciones web, nuestra propuesta se centra principalmente en la construcción de un dispositivo físico, utilizando hardware electrónico como Arduino, sensores y módulos de reconocimiento de voz.

El objetivo es crear una solución tecnológica que permita la interacción directa con el usuario sin necesidad de interfaces digitales externas.

---

## Justificación del Enlace

La actividad solicita el enlace de una aplicación en desarrollo. Sin embargo, nuestro proyecto no requiere una aplicación digital independiente, ya que la interacción con el usuario se realiza directamente mediante hardware y reconocimiento de voz integrado en el dispositivo.

Por esta razón, este repositorio funciona como evidencia digital del desarrollo del proyecto, incluyendo:

- Planeación del sistema  
- Investigación previa  
- Arquitectura del dispositivo  
- Recursos y tutoriales base  
- Avances del diseño  

---

## Estado Actual del Proyecto

Actualmente el proyecto se encuentra en fase de planificación y diseño, debido a que aún no se dispone del hardware necesario para iniciar la programación física del sistema.

En esta etapa se están definiendo los componentes, la arquitectura electrónica y la lógica de funcionamiento general.
